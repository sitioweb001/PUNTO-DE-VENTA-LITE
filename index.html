Ajustes del Sistema

Apariencia

Modo Mantenimiento

Bloquea el acceso al sistema mostrando una pantalla de mantenimiento. Se desactiva con contraseña (doble clic en la pantalla).

Base de Datos

Inicializar crea las hojas si no existen y sincroniza la estructura. No borra datos existentes.

Validación de Código de Barras

Google Sheets borra los ceros del inicio de un código al guardarlo (ej. 000753081010115 se guarda como 753081010115). El lector/pistola de código de barras, en cambio, siempre envía el código completo con sus ceros. Activa esto para que el buscador (Ventas, Carrito, Inventario, Compras, Editor de hoja) ignore los ceros a la izquierda que sean necesarios al comparar, sin importar cuántos sean.

Desactivado (búsqueda exacta)

Modo Venta Sin Conexión

Si se va la luz o el internet, el título "Punto de Venta" parpadea en rojo y el sistema permite seguir vendiendo: el producto se anota a mano y la venta se guarda en este dispositivo hasta que vuelva la conexión, momento en el que se verifica contra la base de datos real (buscador, precio y stock reales) antes de registrarla. Desactiva esto si no quieres que el sistema muestre ese aviso ni permita ventas sin conexión.
